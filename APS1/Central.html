Pequena descrição do que aprendemos>

Por meio das aulas do professor Matheus, foi possível aprender sobre HTML, CSS e a hospedagem de sites no GitHub Pages. HTML (HyperText Markup Language) é a linguagem que define a estrutura de uma página web, como títulos, parágrafos e imagens, funcionando como o esqueleto do site. CSS (Cascading Style Sheets) é responsável pelo visual, permitindo estilizar cores, fontes, layouts e outros detalhes estéticos, dando vida ao design da página. Além disso, aprendi a hospedar sites de minha autoria no GitHub Pages, um serviço gratuito do GitHub que publica páginas web a partir de arquivos HTML, CSS e outros, armazenados em um repositório online.
